### Client 端
* 配置文件 /etc/shadowsocks.json. Example:
```
{
    "server":"45.77.102.215",
    "server_port":8989,
    "local_address": "127.0.0.1",
    "local_port":1080,
    "password":"mypassword",
    "timeout":300,
    "method":"aes-256-gcm,
    "fast_open": false
}
```
* 启动脚步：
```
sudo sslocal -c /etc/shadowsocks.json -d start
```
* 停止脚步：
```
sudo sslocal -c /etc/shadowsocks.json -d stop
```
